## Application Overview
This application facilitates the generation of PDF invoices from a CSV file. Users upload a CSV containing customer data and provide additional invoice details through a web form. The system then processes this information to generate individual PDF invoices and provides them for download.

## Key Functionalities
- CSV File Upload: Users can upload a CSV file that serves as the primary data source for invoice generation.
- Invoice Detail Input: A web form allows users to specify essential invoice details such as date, contract type, starting invoice number, concept, quantity, rate, and taxes.
- PDF Invoice Generation: The application processes the uploaded CSV data and provided invoice details to create individual PDF invoices.
- Job Status Tracking: Upon submission, a unique job ID is provided, and the application displays the processing status.- Invoice Download: Once processing is complete, users can download a ZIP archive containing all generated PDF invoices.
